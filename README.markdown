# phpWowI18N

Easy management Internationalization and Localization

## Requirement

PHP 5.3+

## Usage

### Work with Composer

#### Edit `composer.json`

```
{
    "require": {
        "yftzeng/wow-i18n": "dev-master"
    }
}
```

#### Update composer

```
$ php composer.phar update
```

#### Sample code

Checkout the `samples` dir.
